## Admin-панель (frontend)

Это минимальный frontend админ-панели на **React + Vite + TypeScript**.

### Установка

```bash
cd verdikt-gpt-603a3ce27eb9cc272627d6a213cb93645123d432
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

Откройте приложение в браузере по адресу `http://localhost:5173`.

### Содержимое

- **Дашборд**: карточки со статистикой и таблица последних действий.
- **Пользователи**: таблица пользователей и кнопка создания нового пользователя (пока без логики).

Данные сейчас захардкожены в коде — сюда легко подключить реальные API-запросы.
