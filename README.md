## Verdikt GPT + Админ‑панель

Проект представляет собой локальный веб‑чат **Verdikt GPT** и встроенную админ‑панель.

### Как запустить

- Откройте файл `index.html` в браузере (двойной клик или через «Открыть с помощью...»).
- На верхней панели есть кнопка **«Админ режим»** (иконка инструмента) — по клику вы перейдёте в админ‑панель.

### Структура

- `index.html` — основной интерфейс Verdikt GPT (чат).
- `admin-embedded.html` — страница админ‑панели, открывается по кнопке «Админ режим».

Никаких `npm install` и серверов не требуется — всё работает как обычный статический сайт.
